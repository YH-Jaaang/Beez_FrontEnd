# Beez_FrontEnd

# 개요
```bash
과학기술정보통신부가 주최한 혁신성장 블록체인 시스템 엔지니어 양성과정 프로젝트로
이더리움 기반 지역상권 선순환 지역화폐 시스템 입니다.
```
